    About CiviqPulse

    Youth-led research for clearer civic decisions

    CiviqPulse turns local observations, resident feedback, and structured field data into practical dashboards that help communities understand where attention is needed most.

    Mission

    Data with purpose

    We study everyday civic issues and organize findings into evidence that students, residents, and decision-makers can understand quickly.

    Current focus

    Brampton parks

    The active project reviews park cleanliness across postal-code zones using field visits, a consistent rating rubric, and check-in records.

    Method

    Repeatable research

    Each project is designed so new topics can reuse the same structure: question, dataset, field evidence, analysis, dashboard, and outreach.

    Student work

    Built to grow

    This portfolio supports the current Grade 11 research project and is prepared for future investigations such as employment, food access, and walkability.

    How we work

    CiviqPulse starts with a civic question, collects local evidence, and presents the findings in a clear dashboard.
    • Use consistent rubrics so locations can be compared fairly.
    • Keep data in source files so updates do not require rebuilding the website.
    • Highlight the method and evidence behind each conclusion.
    • Share results in a format that supports discussion and follow-up.
